# Jogo da velha
Tentativa de crirar um tradicional jogo da velha com código python.

## Objetivo
O meu objetivo ao crirar este jogo não foi fazer algo extraordinário, mas sim me refamiliarizar com a linguagem de programação Python que não praticava a tempos.

## O jogo
De maneira breve posso dizer que o jogo, assim como seu código é bastante simples e é feito para se jogar no próprio terminal que você rodar o jogo.
